## what is volt?

_name_ is tbh; volt is a analytics tool for your voice ai applications.

you can proxy your calls and not only log them, but also understand what went right and wrong in every call.



## setting up volt

<Warning>
  volt is in `alpha` - everything is subject to change and most features are not available yet.
</Warning>

### first class support

built in, easy to use support for frameworks

<Icon icon="check" iconType="thin" /> openai realtime wss

<Icon icon="x" iconType="thin" /> openai typescript sdk (coming soon)

<Icon icon="x" iconType="thin" /> openai python sdk (coming soon)

<Icon icon="x" iconType="thin" /> vapi (coming soon)

<Icon icon="x" iconType="thin" /> openai livekit (coming soon)

some of these frameworks might work out of the box, but not tested unless mentioned above.



## basic example usage

this is an example using default (`in-secure`) proxy mode, to learn more about proxy modes check [here](/proxy-modes)

<Steps>
  <Step title="setup account">
    head over to [volt.bench.audio](https://volt.bench.audio) and sign up with github for free.
  </Step>
  <Step title="get project/api key">
    go to your project -> settings -> api keys and create a new key. you will need both your project id and api key in order to use volt.
    <Frame caption="your project settings page">
      <img src="/images/settings.jpeg" />
    </Frame>
  </Step>
  <Step title="change wss url to proxy url">
  point your application to volt's proxy url.
   ```
   https://wss.volt.bench.audio/p/{projectId}?voltToken=v_tok_{apiKey}
   ```


 

   if you are using [openai-realtime-console](https://github.com/openai/openai-realtime-console?tab=readme-ov-file), you will have to make these slight changes you can see them [here](/examples/openai-realtime-console)



   

  that's it, you should see your calls in the dashboard now!
  </Step>
</Steps>
